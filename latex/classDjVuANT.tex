\doxysubsection{Detailed Description}
This class contains some trivial annotations of the page or of the document such as page border color, page alignment, initial zoom and display mode, hyperlinks and highlighted areas. All this information is put inside a textual chunk \#\+A\+N\+Ta\# in pseudo-\/lisp format. Decoding and encoding are normally done by \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuANT_a9d0e8a0e0e1a8fe7f8b8799a68d1da24}{Dj\+Vu\+A\+N\+T\+::decode}}\}() and \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuANT_ab8998d9c990d9e6db6feb1b72486c8f6}{Dj\+Vu\+A\+N\+T\+::encode}}\}() functions.
