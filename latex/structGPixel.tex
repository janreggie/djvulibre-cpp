\doxysubsection{Detailed Description}
Color pixel as a R\+GB triple. ~\newline
 The colors are represented using three bytes named \mbox{\hyperlink{structGPixel_a4464f662afa439bfacbd4f49fd986333}{r}}\#, \mbox{\hyperlink{structGPixel_a0b7123030699a686eaf318f92dee2d9a}{g}}\# and \mbox{\hyperlink{structGPixel_ad10ecdd0b88ddc56c71d63e032d5ffe5}{b}}\#. The value of these bytes represent additive amounts of light. Color white is represented by setting all three bytes to \#255\#. Color black is represented by setting all three bytes to \#0\#. This convention should not be confused with the convention adopted for class \textbackslash{}\+Ref\{\mbox{\hyperlink{classGBitmap}{G\+Bitmap}}\} where the pixel values represent an ink level. ~\newline
