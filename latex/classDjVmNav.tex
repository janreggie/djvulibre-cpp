\doxysubsection{Detailed Description}
The N\+A\+VM chunk. The optional \#\char`\"{}\+N\+A\+V\+M\char`\"{}\# chunk which follows the D\+I\+RM chunk describes how the user can navigate the document. This is a list of Dj\+Vu\+Book\+Marks.

The Dj\+Vu\+Book\+Mark. Each entry in the Navigation chunk (N\+A\+VM) is a bookmark. A bookmark contains a count of immediate children, a display string and a url.
