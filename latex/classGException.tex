\doxysubsection{Detailed Description}
Exception class. ~\newline
 The library always uses macros \#\+G\+\_\+\+T\+RY\#, \#\+G\+\_\+\+T\+H\+R\+OW\#, \#\+G\+\_\+\+C\+A\+T\+CH\# and \#\+G\+\_\+\+E\+N\+D\+C\+A\+T\+CH\# for throwing and catching exceptions (see \textbackslash{}\+Ref\{\mbox{\hyperlink{GException_8h_source}{G\+Exception.\+h}}\}). These macros only deal with exceptions of type \mbox{\hyperlink{classGException}{G\+Exception}}\#.

These strings can be tested by the exception handlers, with \mbox{\hyperlink{classGException_aad6e3fc7bab9b07372366ffa56154693}{cmp\+\_\+cause}}\#.
