\doxysubsection{Detailed Description}
Base class for \mbox{\hyperlink{classGBitmapScaler}{G\+Bitmap\+Scaler}} and \mbox{\hyperlink{classGPixmapScaler}{G\+Pixmap\+Scaler}}. This base class implements the common elements of class \textbackslash{}\+Ref\{\mbox{\hyperlink{classGBitmapScaler}{G\+Bitmap\+Scaler}}\} and \textbackslash{}\+Ref\{\mbox{\hyperlink{classGPixmapScaler}{G\+Pixmap\+Scaler}}\}. Functions \textbackslash{}\+Ref\{set\+\_\+input\+\_\+size\} and \textbackslash{}\+Ref\{set\+\_\+output\+\_\+size\} are used to specify the size of the input image and the size of the output image. Functions \textbackslash{}\+Ref\{set\+\_\+horz\+\_\+ratio\} and \textbackslash{}\+Ref\{set\+\_\+vert\+\_\+ratio\} may be used to override the scaling ratios computed from the image sizes. You can then call function \textbackslash{}\+Ref\{get\+\_\+input\+\_\+rect\} to know which pixels of the input image are necessary to compute a specified rectangular zone of the output image. The actual computation is then performed by calling function \#scale\# in class \textbackslash{}\+Ref\{\mbox{\hyperlink{classGBitmapScaler}{G\+Bitmap\+Scaler}}\} and \textbackslash{}\+Ref\{\mbox{\hyperlink{classGPixmapScaler}{G\+Pixmap\+Scaler}}\}. ~\newline
