\doxysubsection{Detailed Description}
Thread class. A multithreaded process is composed of a main execution thread and of several secondary threads. Each secondary thread is represented by a \mbox{\hyperlink{classGThread}{G\+Thread}}\# object. The amount of memory required for the stack of a secondary thread is defined when the \mbox{\hyperlink{classGThread}{G\+Thread}}\# object is constructed. The execution thread is started when function \textbackslash{}\+Ref\{\mbox{\hyperlink{classGThread_a6fbfcb03fe19741152e9a3d3a55490cb}{G\+Thread\+::create}}\} is called. The destructor of class \mbox{\hyperlink{classGThread}{G\+Thread}} waits until the thread terminanes. Note that the execution can be terminated at any time (with possible prejudice) by calling \textbackslash{}\+Ref\{\mbox{\hyperlink{classGThread_a92dc830b2947d6ce82f26c17f8e10f63}{G\+Thread\+::terminate}}\}.

Several static member functions control the thread scheduler. Function \textbackslash{}\+Ref\{\mbox{\hyperlink{classGThread_afaab53579229a1731ed6bc3eb95481d0}{G\+Thread\+::yield}}\} relinquishes the processor to another thread. Function \textbackslash{}\+Ref\{G\+Thread\+::select\} (\#\+C\+O\+T\+H\+R\+E\+A\+DS\# only) provides a thread-\/aware replacement for the well-\/known unix system call \#select\#. ~\newline


\{\textbackslash{}bf Note\} --- Both the copy constructor and the copy operator are declared as private members. It is therefore not possible to make multiple copies of instances of this class, as implied by the class semantic.

\doxysubsubsection{\texorpdfstring{yield()}{yield()}}
{\footnotesize\ttfamily static int G\+Thread\+::yield (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Causes the current thread to relinquish the processor. The scheduler selects a thread ready to run and transfers control to that thread. The actual effect of \mbox{\hyperlink{classGThread_afaab53579229a1731ed6bc3eb95481d0}{yield}}\# heavily depends on the selected implementation. Function \mbox{\hyperlink{classGThread_afaab53579229a1731ed6bc3eb95481d0}{yield}}\# usually returns zero when the execution of the current thread is resumed. It may return a positive number when it can determine that the current thread will remain the only runnable thread for some time. You may then call function \textbackslash{}\+Ref\{get\+\_\+select\} to obtain more information.
