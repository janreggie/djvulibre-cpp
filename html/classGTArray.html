Detailed Description

template<class TYPE>
class GTArray< TYPE >

Dynamic array for simple types.
Template class #GTArray<TYPE># implements an array of elements of {simple} type #TYPE#. {Simple} means that objects of type #TYPE# can be created, copied, moved or destroyed without using specific constructors or destructor functions. Class #GTArray<TYPE># will move or copy objects using simple bitwise copies. Otherwise you must use class #GArray<TYPE>#. This class only implement constructors. See class \Ref{GArrayTemplate} for a description of all access methods.
