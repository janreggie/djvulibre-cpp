\doxysubsection{Detailed Description}
This is the base object for all map areas. It defines some standard interface to access the geometrical properties of the areas and describes the area itsef\+: \textbackslash{}begin\{itemize\} \textbackslash{}item \mbox{\hyperlink{classGMapArea_ad0a56b6bd099852cb444fc705fa91864}{url}}\# If the optional \#\+U\+RL\# is specified, the map area will also work as a hyperlink meaning that if you click it with your mouse pointer, the browser will be advised to load the page referenced by the \#\+U\+RL\#. \textbackslash{}item \mbox{\hyperlink{classGMapArea_aea992b37a3d1fdb8665c5c5c129be4e9}{target}}\# Defines where the specified \#\+U\+RL\# should be loaded \textbackslash{}item \mbox{\hyperlink{classGMapArea_aad3364122ef1e274838abbb72d91af25}{comment}}\# This is a string displayed in a status line or in a popup window when the mouse pointer moves over the hyperlink area \textbackslash{}item \mbox{\hyperlink{classGMapArea_a2b12ed98955212d4a8c9abf062216569}{border\+\_\+type}}\#, \mbox{\hyperlink{classGMapArea_aed297d702f1082df48257f44d7887eaa}{border\+\_\+color}}\# and \mbox{\hyperlink{classGMapArea_a473b46580d92ea34a14d79b8108dae3a}{border\+\_\+width}}\# describes how the area border should be drawn \textbackslash{}item \#area\+\_\+color\# describes how the area should be highlighted. \textbackslash{}end\{itemize\}

The map areas can be displayed using two different techniques, which can be combined together\+: \textbackslash{}begin\{itemize\} \textbackslash{}item Visible border. The border of a map area can be drawn in several different ways (like \#\+X\+O\+R\+\_\+\+B\+O\+R\+D\+ER\# or \#\+S\+H\+A\+D\+O\+W\+\_\+\+I\+N\+\_\+\+B\+O\+R\+D\+ER\#). It can be made always visible, or appearing only when the mouse pointer moves over the map area. \textbackslash{}item Highlighted contents. Contents of rectangular map areas can also be highlighted with some given color. \textbackslash{}end\{itemize\}
