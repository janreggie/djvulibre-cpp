\doxysubsection{Detailed Description}
Abstract class for a stream of bytes. \mbox{\hyperlink{structClass}{Class}} \mbox{\hyperlink{classByteStream}{Byte\+Stream}}\# represent an object from which (resp. to which) bytes can be read (resp. written) as with a regular file. Virtual functions \mbox{\hyperlink{classByteStream_ae69bbdbc77545f72488bde0639ae0e95}{read}}\# and \mbox{\hyperlink{classByteStream_ac10e950d655e5dabb93af44e3a5da855}{write}}\# must implement these two basic operations. In addition, function \mbox{\hyperlink{classByteStream_a6379ec44f814a160980d2dd5c5e25672}{tell}}\# returns an offset identifying the current position, and function \mbox{\hyperlink{classByteStream_a601ef5bf4d183b262472e1e1f7c9a2ad}{seek}}\# may be used to change the current position.

\{\textbackslash{}bf Note\}. Both the copy constructor and the copy operator are declared as private members. It is therefore not possible to make multiple copies of instances of this class, as implied by the class semantic. ~\newline


Byte\+Stream\+::\+Wrapper implements wrapping bytestream. This is useful for derived classes that take a \mbox{\hyperlink{classGP}{G\+P$<$\+Byte\+Stream$>$}} as a creation argument, and the backwards compatible bytestreams. ~\newline
