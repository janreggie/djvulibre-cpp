Detailed Description

template<class TYPE>
class GList< TYPE >

Doubly linked lists. Template class GList<TYPE># implements a doubly linked list of elements of type #TYPE#. This class only implement constructors. See class \Ref{GListTemplate} and \Ref{GPosition} for a description of all access methods.
