class G\+Map\+Template$<$ K\+T\+Y\+P\+E, V\+T\+Y\+P\+E, T\+I $>$}

Common base class for all associative maps. \mbox{\hyperlink{structClass}{Class}} \textbackslash{}\+Ref\{\mbox{\hyperlink{classGArrayTemplate}{G\+Array\+Template}}\} implements all methods for manipulating associative maps with key type \#\+K\+T\+Y\+PE\# and value type \#\+V\+T\+Y\+PE\#. You should not however create instances of this class. You should instead use class \textbackslash{}\+Ref\{\mbox{\hyperlink{classGMap}{G\+Map}}\} or \textbackslash{}\+Ref\{\mbox{\hyperlink{classGPMap}{G\+P\+Map}}\}.
