\doxysubsection{Detailed Description}
R\+GB Color images. ~\newline
 Instances of class \mbox{\hyperlink{classGPixmap}{G\+Pixmap}}\# represent color images as a two dimensional array of pixels \textbackslash{}\+Ref\{\mbox{\hyperlink{structGPixel}{G\+Pixel}}\}. The bracket operator returns a pointer to the pixels composing one line of the image. This pointer can be used as an array to read or write the pixels of this particular line. Following the general convention of the Dj\+Vu Reference Library, line zero is always the bottom line of the image.
