Detailed Description

A thread safe class representing a set of flags. The flags are protected by \Ref{GMonitor}, which is attempted to be locked whenever somebody accesses the flags. One can modify the class contents using one of two functions: \Ref{test_and_modify}() and \Ref{wait_and_modify}(). Both of them provide atomic operation of testing (first) and modification (second). The flags remain locked between the moment of testing and modification, which guarantees, that their state cannot be changed in between of these operations.
