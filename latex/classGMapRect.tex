\doxysubsection*{G\+Map\+Areas.\+h}
\label{_amgrp7f4144df4046c1505fd4806c7e1d347a}%
Files \#\char`\"{}\+G\+Map\+Areas.\+h\char`\"{}\# and \#\char`\"{}\+G\+Map\+Areas.\+cpp\char`\"{}\# implement base objects used by the plugin to display and manage hyperlinks and highlighted areas inside a \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuImage}{Dj\+Vu\+Image}}\} page.

The currently supported areas can be rectangular (\textbackslash{}\+Ref\{\mbox{\hyperlink{classGMapRect}{G\+Map\+Rect}}\}), elliptical (\textbackslash{}\+Ref\{\mbox{\hyperlink{classGMapOval}{G\+Map\+Oval}}\}) and polygonal (\textbackslash{}\+Ref\{\mbox{\hyperlink{classGMapPoly}{G\+Map\+Poly}}\}). Every map area besides the definition of its shape contains information about display style and optional \{\textbackslash{}bf U\+RL\}, which it may refer to. If this \{\textbackslash{}bf U\+RL\} is not empty then the map area will work like a hyperlink.

The classes also implement some useful functions to ease geometry manipulations

\doxysubsection{Detailed Description}
Implements rectangular map areas. This is the only kind of map areas supporting \#\+S\+H\+A\+D\+O\+W\+\_\+\+I\+N\+\_\+\+B\+O\+R\+D\+ER\#, \#\+S\+H\+A\+D\+O\+W\+\_\+\+O\+U\+T\+\_\+\+B\+O\+R\+D\+ER\#, \#\+S\+H\+A\+D\+O\+W\+\_\+\+E\+I\+N\+\_\+\+B\+O\+R\+D\+ER\# and \#\+S\+H\+A\+D\+O\+W\+\_\+\+E\+O\+U\+T\+\_\+\+B\+O\+R\+D\+ER\# types of border and area highlighting.
