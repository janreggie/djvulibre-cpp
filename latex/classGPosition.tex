\doxysubsection{Detailed Description}
Generic iterator class. This class represents a position in a list (see \textbackslash{}\+Ref\{\mbox{\hyperlink{classGList}{G\+List}}\}) or a map (see \textbackslash{}\+Ref\{\mbox{\hyperlink{classGMap}{G\+Map}}\}). As demonstrated by the following examples, this class should be used to iterate over the objects contained in a list or a map\+: \textbackslash{}begin\{verbatim\} void print\+\_\+list(\+G\+List$<$\+G\+String$>$ a) \{ for (\mbox{\hyperlink{classGPosition}{G\+Position}} i = a ; i; ++i) Dj\+Vu\+Print\+Message(\char`\"{}\%s\textbackslash{}n\char`\"{}, (const char$\ast$) a\mbox{[}i\mbox{]} ); \}

void print\+\_\+list\+\_\+backwards(\+G\+List$<$\+G\+String$>$ a) \{ for (\mbox{\hyperlink{classGPosition}{G\+Position}} i = a.\+lastpos() ; i; --i) Dj\+Vu\+Print\+Message(\char`\"{}\%s\textbackslash{}n\char`\"{}, (const char$\ast$) a\mbox{[}i\mbox{]} ); \} \textbackslash{}end\{verbatim\} \mbox{\hyperlink{classGPosition}{G\+Position}} objects should only be used with the list or map for which they have been created (using the member functions \#firstpos\# or \#lastpos\# of the container). Furthermore, you should never use a \mbox{\hyperlink{classGPosition}{G\+Position}} object which designates a list element which has been removed from the list (using member function \#del\# or by other means.)
