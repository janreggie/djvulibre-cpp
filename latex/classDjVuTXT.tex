\doxysubsection{Detailed Description}
Description of the text contained in a Dj\+Vu page. This class contains the textual data for the page. It describes the text as a hierarchy of zones corresponding to page, column, region, paragraph, lines, words, etc... The piece of text associated with each zone is represented by an offset and a length describing a segment of a global U\+T\+F8 encoded string. ~\newline
