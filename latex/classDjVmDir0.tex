\doxysubsection{Detailed Description}
Directory of all-\/in-\/one-\/file Dj\+Vu documents (also known as Dj\+Vm documents). This class can read and write the directory (table of contents, in other words) of a Dj\+Vm document. This table of contents lists all \{\textbackslash{}bf files\} included into the document, not \{\textbackslash{}bf pages\} like \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuNavDir}{Dj\+Vu\+Nav\+Dir}}\} does. It is normally stored in the document inside \{\textbackslash{}bf D\+I\+R0\} chunk where \{\textbackslash{}bf \char`\"{}0\char`\"{}\} refers to the version number.

An object of this class can be created either as a result of the decoding of an existing Dj\+Vm file, or manually by calling the default constructor and adding later directory entries by means of \textbackslash{}\+Ref\{add\+\_\+file\}() function.

You normally will not want to create or decode this directory manually. \textbackslash{}\+Ref\{Dj\+Vm\+File\} class will do it for you.
