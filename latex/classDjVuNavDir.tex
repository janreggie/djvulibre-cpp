\doxysubsection{Detailed Description}
Dj\+Vu Navigation Directory.

This class implements the \{{\itshape navigation} directory\} of a multipage Dj\+Vu document -\/ basically a list of pages that this document is composed of. We would like to emphasize, that this is the list of namely \{\textbackslash{}bf pages\}, not \{\textbackslash{}bf files\}. Any page may include any number of additional files. When you\textquotesingle{}ve got an all-\/in-\/one-\/file multipage Dj\+Vu document (Dj\+Vm archive) you may get the files list from \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVmDir0}{Dj\+Vm\+Dir0}}\} class.

The \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuNavDir}{Dj\+Vu\+Nav\+Dir}}\} class can decode and encode the navigation directory from \{\textbackslash{}bf N\+D\+IR\} I\+FF chunk. It\textquotesingle{}s normally created by the library during decoding procedure and can be accessed like any other component of the \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuImage}{Dj\+Vu\+Image}}\} being decoded.

In a typical multipage Dj\+Vu document the navigation directory is stored in a separate I\+FF file containing only one chunk\+: \{\textbackslash{}bf N\+D\+IR\} chunk. This file should be included (by means of the \{\textbackslash{}bf I\+N\+CL\} chunk) into every page of the document to enable the navigation.
