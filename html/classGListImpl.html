Doubly Linked Lists

The template classes \Ref{GList} and \Ref{GPList} implement a doubly linked list of objects of arbitrary types. Member functions are provided to search the list for an element, to insert or delete elements at specified positions. Theses template class must be able to access \begin{itemize} \item a default constructor #TYPE::TYPE()#, \item a copy constructor #TYPE::TYPE(const TYPE &)#, \item optionally a destructor #TYPE::~TYPE()#, \item and optionally a comparison operator #TYPE::operator==(const TYPE &)#. \end{itemize} @memo Doubly linked lists.
