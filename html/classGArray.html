Detailed Description

template<class TYPE>
class GArray< TYPE >

Dynamic array for general types.
Template class GArray<TYPE># implements an array of elements of type #TYPE#. This template class must be able to access the following functions. \begin{itemize} \item a default constructor #TYPE::TYPE()#, \item a copy constructor #TYPE::TYPE(const TYPE &)#, \item and optionally a destructor #TYPE::~TYPE()#. \end{itemize} This class only implement constructors. See class \Ref{GArrayTemplate} for a description of all access methods.
