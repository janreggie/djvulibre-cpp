\doxysubsection{Detailed Description}
\mbox{\hyperlink{classDjVuDocEditor}{Dj\+Vu\+Doc\+Editor}}\# is an extension of \textbackslash{}\+Ref\{\mbox{\hyperlink{classDjVuDocument}{Dj\+Vu\+Document}}\} class with additional capabilities for editing the document contents.

It can be used to\+: \textbackslash{}begin\{enumerate\} \textbackslash{}item Create (compose) new multipage Dj\+Vu documents using single page Dj\+Vu documents. The class does \{\textbackslash{}bf not\} do compression. \textbackslash{}item Insert and remove different pages of multipage Dj\+Vu documents. \textbackslash{}item Change attributes (\{{\itshape names\}}, \{{\itshape I\+Ds\}} and \{{\itshape titles\}}) of files composing the Dj\+Vu document. \textbackslash{}item Generate thumbnail images and integrate them into the document. \textbackslash{}end\{enumerate\}

Returns \#\+T\+R\+UE\# if the document can be \char`\"{}saved\char`\"{} (sometimes the only possibility is to do a \char`\"{}save as\char`\"{}). The reason why we have this function is that \mbox{\hyperlink{classDjVuDocEditor}{Dj\+Vu\+Doc\+Editor}}\# can save documents in new formats only (\#\+B\+U\+N\+D\+L\+ED\# and \#\+I\+N\+D\+I\+R\+E\+CT\#). At the same time it recognizes all Dj\+Vu formats (\#\+O\+L\+D\+\_\+\+B\+U\+N\+D\+L\+ED\#, \#\+O\+L\+D\+\_\+\+I\+N\+D\+E\+X\+ED\#, \#\+B\+U\+N\+D\+L\+ED\#, and \#\+I\+N\+D\+I\+R\+E\+CT\#).

\#\+O\+L\+D\+\_\+\+B\+U\+N\+D\+L\+ED\# and \#\+B\+U\+N\+D\+L\+ED\# documents occupy only one file, so in this case \char`\"{}saving\char`\"{} involves the automatic conversion to \#\+B\+U\+N\+D\+L\+ED\# format and storing data into the same file.

\#\+O\+L\+D\+\_\+\+I\+N\+D\+E\+X\+ED\# documents, on the other hand, occupy more than one file. They could be converted to \#\+I\+N\+D\+I\+R\+E\+CT\# format if these two formats had the same set of files. If you call \textbackslash{}\+Ref\{insert\+\_\+page\}() for every page, this shared file will be inserted into the document more than once though under different names.
