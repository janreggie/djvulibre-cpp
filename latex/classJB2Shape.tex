\doxysubsection{Detailed Description}
Shape data structure. A \mbox{\hyperlink{classJB2Image}{J\+B2\+Image}}\# contains an array of \mbox{\hyperlink{classJB2Shape}{J\+B2\+Shape}}\# data structures. Each array entry represents an elementary blob of ink such as a character or a segment of line art. Member \mbox{\hyperlink{classJB2Shape_abcf57fa6b646612010430eee98672dfb}{bits}}\# points to a bilevel image representing the shape pixels. Member \mbox{\hyperlink{classJB2Shape_a06620dbda5f424ebe8b4218384072595}{parent}}\# is the subscript of the parent shape. ~\newline

Subscript of the parent shape. The parent shape must always be located before the current shape in the shape array. A negative value indicates that this shape has no parent. Any negative values smaller than \#-\/1\# further indicates that this shape does not look like a character. This is used to enable a few internal optimizations.
