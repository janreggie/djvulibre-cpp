class G\+List\+Template$<$ T\+Y\+P\+E, T\+I $>$}

Common base class for all doubly linked lists. ~\newline
 \mbox{\hyperlink{structClass}{Class}} \textbackslash{}\+Ref\{\mbox{\hyperlink{classGListTemplate}{G\+List\+Template}}\} implements all methods for manipulating lists of of objects of type \#\+T\+Y\+PE\#. You should not however create instances of this class.
