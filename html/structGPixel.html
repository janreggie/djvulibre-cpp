Detailed Description

Color pixel as a RGB triple.
The colors are represented using three bytes named r#, g# and b#. The value of these bytes represent additive amounts of light. Color white is represented by setting all three bytes to #255#. Color black is represented by setting all three bytes to #0#. This convention should not be confused with the convention adopted for class \Ref{GBitmap} where the pixel values represent an ink level.
