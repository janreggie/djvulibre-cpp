Detailed Description

template<class KTYPE, class VTYPE, class TI>
class GMapTemplate< KTYPE, VTYPE, TI >

Common base class for all associative maps. Class \Ref{GArrayTemplate} implements all methods for manipulating associative maps with key type #KTYPE# and value type #VTYPE#. You should not however create instances of this class. You should instead use class \Ref{GMap} or \Ref{GPMap}.
